## Tech Stack

**Client:** Next JS and Chakra UI

**Server:** PostgreSQL and Prisma

## Run Locally

Clone the project

```bash
  git clone https://github.com/amr8644/E-commerce
```

Go to the project directory

```bash
  cd E-commerce
```

Install dependencies

```bash
  yarn install
```

Run

```bash
  yarn run dev
```
